# Advanced Learning Algorithms 🚀

This repository contains implementations and explanations of **advanced machine learning algorithms** using **Python** and **scikit-learn**.  

##  Topics which we cover in projects continuesly 

###  Supervised Learning
- Logistic Regression (Advanced usage)
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Decision Tree
- Random Forest
- Naive Bayes (Gaussian, Multinomial, Bernoulli)

###  Unsupervised Learning
- K-Means Clustering
- Hierarchical Clustering
- DBSCAN
- Principal Component Analysis (PCA)
